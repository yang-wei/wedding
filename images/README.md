# Photos

Drop your own photos here using these exact filenames and they'll appear on the
site automatically. Until a file exists, the slot shows an on-brand illustrated
placeholder — so the site always looks complete.

| Filename            | Where it shows                                  | Suggested size / crop        |
|---------------------|-------------------------------------------------|------------------------------|
| `hero.png`          | Hero — the couple cutout                        | Transparent PNG, ~800×1000   |
| `gallery-1.jpg`     | "How it all started" photo strip (1st)          | Portrait, ~500×640           |
| `gallery-2.jpg`     | …strip (2nd)                                     | Portrait, ~500×640           |
| `gallery-3.jpg`     | …strip (3rd)                                     | Portrait, ~500×640           |
| `gallery-4.jpg`     | …strip (4th)                                     | Portrait, ~500×640           |
| `gallery-5.jpg`     | …strip (5th)                                     | Portrait, ~500×640           |
| `gallery-6.jpg`     | …strip (6th)                                     | Portrait, ~500×640           |
| `event-1.jpg`       | Order of events — Monday card                   | Landscape, ~700×460          |
| `event-2.jpg`       | Order of events — Tuesday card                  | Landscape, ~700×460          |
| `event-3.jpg`       | Order of events — Wednesday card                | Landscape, ~700×460          |

Notes:
- `hero.png` looks best as a background-removed cutout (transparent PNG). A
  regular JPG works too — it'll just be a rectangular photo.
- Gallery images are cropped to fill (`object-fit: cover`), so any aspect ratio
  is fine; portrait crops look best.
- No code changes needed — just add the files and refresh.
